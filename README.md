# Ownable Car Washes for FiveM

Purchasable car washes for **QBOX** and **ESX**

---

## Features

- **Ownable Car Washes**  
  Players can buy and manage car washes as a business.

- **Water System**  
  Each car wash consumes water per use.

- **Delivery System**  
  Owners or employees can start a water delivery job.

- **Creator**  
  Authorized users can create car washes.
---

## Dependencies

- ESX or QBOX
- [ox_lib](https://github.com/overextended/ox_lib/releases)
- [ox_target](https://github.com/overextended/ox_target/releases)
- [ox_inventory](https://github.com/overextended/ox_inventory/releases)
